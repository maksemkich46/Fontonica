# Fontonica - кастомные шрифты для Roblox на телефонах

![](logo.png)

> [!IMPORTANT]
> Выпускаются новые версии Roblox только тогда, когда это необходимо. Когда разработчик видит плашку, с необходимостью обновить Roblox в магазине приложений, то тогда разработчик бежит делать и публиковать новую версию.

> ## Доступные шрифты:
> **Moscow Sans**
